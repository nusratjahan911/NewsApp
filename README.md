## 📰 News App

A simple and modern Flutter News App that fetches the latest news using API integration.
Users can browse trending headlines, read full articles, and explore news by categories.

## 📱 Features
 - 📰 Latest news headlines
- 🔍 Browse news by categories
- 🌐 API integration for real-time news
- 📖 Read full articles in WebView
- ⚡ Fast and responsive UI
- 📱 Clean Flutter design

## 🚀 Technologies Used
- Flutter
- Dart
- REST API
- HTTP package
- WebView Flutter
  
## 🔗 API Used
 This app uses a News API to fetch live news data.
- Example:
https://newsapi.org/
